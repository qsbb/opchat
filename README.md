# OpChat

将原版 Minecraft 聊天框替换为 QQ / 微信风格的聊天气泡界面。

A Fabric mod that replaces the vanilla Minecraft chat screen with a QQ/WeChat-style chat bubble UI.

## 功能特性

- **聊天气泡** — HUD 上显示带头像、名称、时间的聊天气泡，带渐变消失动画
- **点击交互** — 点击玩家头像 @提及，右键气泡弹出菜单（复制 / 引用回复）
- **引用回复** — 引用他人消息并在历史中高亮显示被引用的内容
- **消息预览** — 聊天图标上方显示最新消息预览（可配置行数与宽度）
- **强提示** — 被 @ 或被引用时放大加粗显示并伴随音效；系统消息同样可强提示
- **未读红点** — 聊天图标上的红点提示，被 @ 时变为更醒目的颜色
- **防刷屏** — 合并连续相同的短消息为 `xN`
- **快捷输入** — 点击 ⚡ 按钮弹出快捷输入面板，支持添加 / 删除条目，点击直接发送（`/` 开头视为命令），持久化保存
- **快捷指令** — 点击 `/` 按钮打开快捷指令面板；按键盘 `/` 可像原版一样预输入 `/` 并弹出命令补全
- **历史召回** — 使用上下方向键召回发送历史，并可在输入框上方显示历史发送记录预览
- **联系人分组** — 支持联系人分组管理，分组成员编辑可展开 / 收起，支持 Enter 添加与 Delete 删除
- **面板透明度** — 0–100 可调，适配不同视觉偏好
- **自定义标题** — 点击标题旁的笔形图标可编辑当前世界 / 服务器的聊天标题
- **命令补全** — 输入 `/` 自动弹出命令补全窗口
- **配置屏幕** — 点击齿轮图标打开可视化配置面板
- **多语言** — 内置中文与英文
- **兼容性** — 可选「禁用聊天举报」模组兼容模式

## 环境要求

- Minecraft 1.21.1（兼容 1.21.11）
- Fabric Loader >= 0.16.0
- Fabric API
- Java >= 21

## 安装

1. 安装 [Fabric Loader](https://fabricmc.net/use/installer/)
2. 下载 [Fabric API](https://modrinth.com/mod/fabric-api) 放入 `mods` 文件夹
3. 下载本模组 JAR 文件放入 `mods` 文件夹

## 操作说明

| 操作 | 功能 |
|------|------|
| `T` / `Enter` | 打开聊天气泡输入框 |
| `/` | 像原版一样打开聊天并预输入 `/`，显示命令建议 |
| `↑` / `↓` | 召回发送历史；开启“历史发送记录预览”时显示可视化列表 |
| 点击齿轮图标 | 打开配置屏幕 |
| 点击 ⚡ 图标 | 打开 / 关闭快捷输入面板 |
| 点击 `/` 图标 | 打开 / 关闭快捷指令面板 |
| 点击发送图标 | 发送当前输入框内容 |
| 点击玩家头像 | @提及该玩家 |
| 右键气泡 | 弹出菜单（复制 / 引用回复） |
| 点击标题笔图标 | 编辑当前世界 / 服务器的聊天标题 |
| 鼠标滚轮 | 滚动查看历史消息 |

## 配置文件

- 配置文件：`config/opchat-client.json`
- 自定义标题：`<游戏目录>/opchat/titles.json`
- 历史发送记录预览：设置 → 私聊与历史 → “历史发送记录预览”

## 构建

```bash
# 需要 Java 21
./gradlew build
```

生成的 JAR 文件在 `build/libs/` 目录下。
